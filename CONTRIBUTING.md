# Contribution guide

Hi there! Thanks for taking the time to read this. 


## Scope

This repo, `anevern-public`, contains the code for my personal artist website, built using [Mintlify](https://www.mintlify.com/). 

Contributions I will accept:
* Spelling and grammar errors
* Accessability improvements
* Improvements to `style.css`
* Technical or functional improvements

Contributions that will not be accetpted:
* Revisions to tone or writing style
* Changing to AP Style
* Alterations to my art (style, themes, subject matter, etc.)
* Drastic changes to visual aesthetic (e.g., changing the whole site to light mode)

## How to Contribute

* Check the [Style Guide](https://github.com/KytheraOA/anevern-public/blob/main/README.md#style-guide) first.
* Discussions: If you have questions, comments, or suggestions you'd like to discuss, start a new discussion: https://github.com/KytheraOA/anevern-public/discussions
* Issues: Please open a new issue and assign it to [@KytheraOA](https://github.com/KytheraOA) here: https://github.com/KytheraOA/anevern-public/issues 
  * Use the `bug` label for site bugs (typos, broken links, missing images, etc.)
  * Use the `accessibility` label for anything related to accessibility. Suggestions for improving accessibility are more than welcome! I am doing my best, and recognize that I have a lot yet to learn.
  * For all other issues, please use your best judgment and use the lables that make the most sense.

Ultimately...just be cool. 😎

### Changes

Preferred method: 
1. Check [Issues](https://github.com/KytheraOA/anevern-public/issues) to see if there is already a related report or [Discussions](https://github.com/KytheraOA/anevern-public/discussions) to see if it's already being talked about
2. If there are no related reports or discussions, open a new issue and add relevant tags (or start a new discussion).

Local changes:
1. At this time, please do not. Open an issue or start a discussion. If forking the repo and creating a pull request are warranted, we'll talk about it. 😅 If this changes, I will update this document.
